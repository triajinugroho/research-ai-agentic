## 3. Detail Kuis (K1–K3)

---

### K1: Kuis Fondasi (Minggu 4)

| Aspek | Detail |
|-------|--------|
| **Cakupan Materi** | Minggu 1–4: Pengantar algoritma dan pemikiran komputasional, variabel, tipe data, operator, input/output, struktur seleksi (if/elif/else), struktur perulangan (for, while) |
| **Durasi** | 30 menit |
| **Format** | Closed-book, tanpa akses komputer atau AI |
| **Jumlah Soal** | 10 soal |
| **Tipe Soal** | Tracing kode (tentukan output), tulis pseudocode untuk masalah sederhana, prediksi output dari potongan kode, identifikasi dan perbaiki error |
| **Bobot** | 7% dari nilai akhir |

---

### K2: Kuis Modularitas (Minggu 7)

| Aspek | Detail |
|-------|--------|
| **Cakupan Materi** | Minggu 5–7: Fungsi (definisi, parameter, return value, scope), string processing, list dan tuple (operasi dasar, slicing, iterasi) |
| **Durasi** | 30 menit |
| **Format** | Closed-book, tanpa akses komputer atau AI |
| **Jumlah Soal** | 10 soal |
| **Tipe Soal** | Tracing fungsi (tentukan return value), tulis fungsi untuk masalah tertentu, operasi string dan list, identifikasi scope variable |
| **Bobot** | 7% dari nilai akhir |

---

### K3: Kuis Algoritma (Minggu 12)

| Aspek | Detail |
|-------|--------|
| **Cakupan Materi** | Minggu 9–12: Dictionary dan set, algoritma pencarian (linear search, binary search), algoritma pengurutan (bubble sort, selection sort, insertion sort), rekursi |
| **Durasi** | 30 menit |
| **Format** | Closed-book, tanpa akses komputer atau AI |
| **Jumlah Soal** | 10 soal |
| **Tipe Soal** | Trace algoritma searching (tentukan langkah-langkah dan hasil), trace algoritma sorting (tunjukkan kondisi array di setiap iterasi), trace fungsi rekursif (tentukan output dan call stack), operasi dictionary dan set |
| **Bobot** | 6% dari nilai akhir |
