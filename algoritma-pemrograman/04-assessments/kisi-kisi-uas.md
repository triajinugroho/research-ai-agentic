### 5.2 Contoh Soal Code Tracing

**Soal CT-1 (Dictionary & Set):**
Tuliskan output dari kode berikut:
```python
suara = ["Roti", "Nasi", "Roti", "Mie", "Nasi", "Roti", "Nasi"]
hitungan = {}
for item in suara:
    if item in hitungan:
        hitungan[item] += 1
    else:
        hitungan[item] = 1

for makanan, jumlah in hitungan.items():
    if jumlah >= 2:
        print(f"{makanan}: {jumlah} suara")

unik = set(suara)
print(f"Total jenis: {len(unik)}")
```

**Jawaban:**
```
Roti: 3 suara
Nasi: 3 suara
Total jenis: 3
```

**Soal CT-2 (Rekursi):**
Tuliskan output dari kode berikut:
```python
def mystery(n):
    if n <= 0:
        return 0
    elif n == 1:
        return 1
    else:
        return mystery(n - 1) + mystery(n - 2)

for i in range(6):
    print(mystery(i), end=" ")
print()
```

**Jawaban:**
```
0 1 1 2 3 5
```

### 5.3 Contoh Soal Tulis Kode

**Soal TK-1 (Dictionary + Fungsi):**
Buatlah program untuk mengelola data inventaris warung. Program harus memiliki:
- Fungsi `tambah_barang(inventaris, nama, harga, stok)` yang menambahkan barang ke dictionary
- Fungsi `cari_barang(inventaris, nama)` yang mengembalikan info barang atau pesan "Barang tidak ditemukan"
- Fungsi `total_nilai_inventaris(inventaris)` yang mengembalikan total nilai seluruh inventaris (harga x stok)

**Contoh Jawaban:**
```python
def tambah_barang(inventaris, nama, harga, stok):
    inventaris[nama] = {"harga": harga, "stok": stok}

def cari_barang(inventaris, nama):
    if nama in inventaris:
        info = inventaris[nama]
        return f"{nama}: Rp{info['harga']}, Stok: {info['stok']}"
    return "Barang tidak ditemukan"

def total_nilai_inventaris(inventaris):
    total = 0
    for barang in inventaris.values():
        total += barang["harga"] * barang["stok"]
    return total
```

**Soal TK-2 (Searching + Algoritma):**
Buatlah fungsi `binary_search(data, target)` yang menerima list terurut dan target, kemudian mengembalikan indeks target jika ditemukan atau -1 jika tidak ditemukan. Implementasikan secara iteratif (tanpa rekursi).

**Soal TK-3 (File I/O + Struktur Data):**
Buatlah dua fungsi:
- `simpan_nilai(nama_file, data_nilai)` yang menerima nama file dan dictionary `{nama: nilai}` lalu menyimpannya ke file CSV
- `baca_nilai(nama_file)` yang membaca file CSV tersebut dan mengembalikan dictionary

### 5.4 Contoh Soal Essay/Analisis

**Soal E-1 (Analisis Big-O + Perbandingan Algoritma):**
Perhatikan dua potongan kode berikut:

**Kode A:**
```python
def cari_a(data, target):
    for i in range(len(data)):
        if data[i] == target:
            return i
    return -1
```

**Kode B:**
```python
def cari_b(data, target):
    low, high = 0, len(data) - 1
    while low <= high:
        mid = (low + high) // 2
        if data[mid] == target:
            return mid
        elif data[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    return -1
```

a) Tentukan kompleksitas waktu Big-O untuk masing-masing kode dalam kasus terbaik (*best case*) dan kasus terburuk (*worst case*). (4 poin)
b) Jelaskan dalam kondisi apa Kode A lebih baik digunakan daripada Kode B, dan sebaliknya. (3 poin)
c) Jika data berisi 1.000.000 elemen, estimasikan jumlah perbandingan maksimal yang diperlukan masing-masing kode. (3 poin)

**Soal E-2 (Etika AI dan Evaluasi Output AI):**
Seorang mahasiswa diminta membuat fungsi untuk menghitung rata-rata nilai mahasiswa. Ia menggunakan AI (ChatGPT) dan mendapatkan kode berikut:

```python
def hitung_rata_rata(nilai_list):
    return sum(nilai_list) / len(nilai_list)
```

a) Identifikasi potensi masalah atau bug dalam kode yang dihasilkan AI tersebut. (3 poin)
b) Tuliskan versi perbaikan kode tersebut yang menangani edge cases. (4 poin)
c) Menurut Anda, bagaimana seharusnya mahasiswa mendokumentasikan penggunaan AI untuk tugas ini dalam AI Usage Log? Jelaskan minimal 3 aspek yang harus dicatat. (3 poin)
