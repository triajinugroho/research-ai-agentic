## 2. Detail Laporan Praktikum (L1-L14)

Laporan praktikum merupakan dokumentasi kegiatan praktikum mingguan yang dikerjakan oleh mahasiswa selama sesi lab. Setiap laporan dikumpulkan dalam bentuk **Google Colab notebook (.ipynb)** melalui Google Classroom/LMS. Batas pengumpulan adalah **H+3 setelah sesi praktikum**.

---

### L1: Setup Python & Google Colab (Minggu 1)

**CPMK:** CPMK-1

**Deskripsi:**
Mahasiswa melakukan setup lingkungan pemrograman Python menggunakan Google Colab. Kegiatan meliputi: membuat akun Google (jika belum), membuat notebook baru, mengenal antarmuka Colab (cell code vs text), menulis dan menjalankan program pertama ("Hello, World!"), serta bereksperimen dengan operasi aritmatika dasar di Colab.

**Deliverable:** Google Colab notebook (.ipynb) berisi program pertama dan eksplorasi antarmuka Colab.

**Rubrik:** Dinilai berdasarkan kelengkapan eksplorasi, keberhasilan menjalankan program, dan kerapian notebook.

---

### L2: Variabel & Ekspresi (Minggu 2)

**CPMK:** CPMK-2

**Deskripsi:**
Mahasiswa membuat program kalkulator sederhana yang mendemonstrasikan penggunaan variabel, tipe data (int, float, string, boolean), operator aritmatika, serta konversi tipe data (type casting). Praktikum mencakup latihan input/output dan pemformatan output.

**Deliverable:** Google Colab notebook (.ipynb) berisi latihan variabel, ekspresi, dan program kalkulator sederhana.

**Rubrik:** Dinilai berdasarkan kebenaran program, pemahaman tipe data, dan penggunaan type casting.

---

### L3: Conditional Logic (Minggu 3)

**CPMK:** CPMK-3

**Deskripsi:**
Mahasiswa membuat program-program keputusan (decision programs) menggunakan struktur seleksi if/elif/else. Latihan meliputi: program penentuan grade nilai, kalkulator BMI dengan kategori, dan program rekomendasi sederhana berdasarkan input pengguna.

**Deliverable:** Google Colab notebook (.ipynb) berisi minimal 3 program yang menggunakan conditional logic.

**Rubrik:** Dinilai berdasarkan kebenaran logika kondisional, penanganan edge cases, dan variasi penggunaan if/elif/else.

**Deliverable:** Google Colab notebook (.ipynb) berisi minimal 4 pola berbeda dan program tebak angka.
