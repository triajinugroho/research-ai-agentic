> **Catatan tentang Unicode dan UTF-8:**
> Tabel ASCII hanya mencakup 128 karakter (0–127). Untuk mendukung karakter internasional (aksara Jawa, Arab, Mandarin, emoji, dll.), digunakan standar **Unicode** yang mencakup lebih dari 140.000 karakter. **UTF-8** adalah encoding Unicode yang paling umum digunakan di web dan sistem modern. Python 3 menggunakan Unicode secara *default* untuk semua string, sehingga Anda dapat langsung menulis `s = "Universitas Al Azhar Indonesia"` atau `s = "Bismillah"` tanpa masalah encoding.
>
> Untuk melihat kode ASCII/Unicode suatu karakter di Python:
> ```python
> ord('A')    # 65
> chr(65)     # 'A'
> ord('Z')    # 90 (huruf latin)
> ```

### B.4 Fungsi

```python
# Definisi fungsi dasar
def nama_fungsi(param1, param2):
    """Docstring: menjelaskan fungsi ini."""
    # blok kode
    return hasil

# Fungsi dengan default parameter
def sapa(nama, sapaan="Halo"):
    """Menyapa seseorang."""
    return f"{sapaan}, {nama}!"

# Fungsi dengan *args dan **kwargs
def fungsi_fleksibel(*args, **kwargs):
    for arg in args:
        print(arg)
    for key, value in kwargs.items():
        print(f"{key} = {value}")

# Lambda (anonymous function)
kuadrat = lambda x: x ** 2
hasil = kuadrat(5)   # 25

# Fungsi sebagai first-class object
def terapkan(fungsi, nilai):
    return fungsi(nilai)

terapkan(kuadrat, 7)  # 49

# Fungsi rekursif
def faktorial(n):
    """Menghitung n! secara rekursif."""
    if n <= 1:          # base case
        return 1
    return n * faktorial(n - 1)   # recursive case
```

### B.9 Algoritma (Pseudocode Quick Reference)

**Linear Search:**
```
FUNGSI linear_search(daftar, target):
    UNTUK setiap elemen pada indeks i:
        JIKA daftar[i] == target:
            KEMBALIKAN i
    KEMBALIKAN -1        # tidak ditemukan
```

```python
def linear_search(data, target):
    for i in range(len(data)):
        if data[i] == target:
            return i
    return -1
```

**Binary Search:**
```
FUNGSI binary_search(daftar_terurut, target):
    low = 0, high = panjang - 1
    SELAMA low <= high:
        mid = (low + high) // 2
        JIKA daftar[mid] == target:
            KEMBALIKAN mid
        JIKA daftar[mid] < target:
            low = mid + 1
        SELAINNYA:
            high = mid - 1
    KEMBALIKAN -1
```

```python
def binary_search(data, target):
    low, high = 0, len(data) - 1
    while low <= high:
        mid = (low + high) // 2
        if data[mid] == target:
            return mid
        elif data[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    return -1
```

**Bubble Sort:**
```python
def bubble_sort(data):
    n = len(data)
    for i in range(n - 1):
        swapped = False
        for j in range(n - 1 - i):
            if data[j] > data[j + 1]:
                data[j], data[j + 1] = data[j + 1], data[j]
                swapped = True
        if not swapped:
            break
    return data
```

**Selection Sort:**
```python
def selection_sort(data):
    n = len(data)
    for i in range(n - 1):
        min_idx = i
        for j in range(i + 1, n):
            if data[j] < data[min_idx]:
                min_idx = j
        data[i], data[min_idx] = data[min_idx], data[i]
    return data
```

**Insertion Sort:**
```python
def insertion_sort(data):
    for i in range(1, len(data)):
        key = data[i]
        j = i - 1
        while j >= 0 and data[j] > key:
            data[j + 1] = data[j]
            j -= 1
        data[j + 1] = key
    return data
```

### B.10 Big-O Quick Reference

| Notasi       | Nama           | Contoh Algoritma                    | Keterangan                              |
|--------------|----------------|-------------------------------------|-----------------------------------------|
| O(1)         | Constant       | Akses array by index, dict lookup   | Waktu tetap, tidak terpengaruh ukuran   |
| O(log n)     | Logarithmic    | Binary search                       | Membagi masalah jadi setengah tiap langkah |
| O(n)         | Linear         | Linear search, traversal list       | Waktu berbanding lurus dengan ukuran    |
| O(n log n)   | Linearithmic   | Merge sort, Timsort (Python sort)   | Sorting efisien                         |
| O(n^2)       | Quadratic      | Bubble sort, Selection sort, nested loop | Tidak efisien untuk data besar     |
| O(2^n)       | Exponential    | Fibonacci rekursif naif             | Sangat lambat, hindari untuk n besar    |

> **Prinsip:** Pilih algoritma dengan Big-O lebih rendah untuk data berukuran besar. Python built-in `sort()` menggunakan Timsort dengan kompleksitas O(n log n).

## Lampiran C: Glosarium Istilah (Bilingual)

| 2  | Akumulator | Accumulator | Variabel yang menyimpan hasil perhitungan kumulatif selama iterasi, misalnya penjumlahan total dalam loop. |

## Lampiran D: Panduan Instalasi Python dan IDE

### D.1 Google Colab (Rekomendasi Utama)

Google Colaboratory (Colab) adalah lingkungan notebook berbasis cloud yang disediakan oleh Google secara gratis. Colab sangat direkomendasikan untuk mata kuliah ini karena tidak memerlukan instalasi apapun di komputer lokal.

**Langkah-langkah mengakses Google Colab:**

1. **Buka browser** (Chrome, Firefox, atau Edge) dan kunjungi: https://colab.research.google.com/
2. **Login dengan akun Google** — Gunakan akun Gmail Anda. Jika belum punya, buat akun di https://accounts.google.com/
3. **Buat notebook baru** — Klik **"New Notebook"** atau pilih **File > New Notebook**.
4. **Mengenal antarmuka Colab:**
   - **Code cell**: Tempat menulis dan menjalankan kode Python. Klik ikon play (atau tekan `Shift+Enter`) untuk menjalankan.
   - **Text cell**: Tempat menulis catatan dalam format Markdown.
   - **Menu bar**: Berisi opsi File, Edit, View, Insert, Runtime, Tools, dan Help.
   - **Toolbar**: Tombol cepat untuk menambah cell, menjalankan kode, dll.
5. **Menulis kode pertama:**
   - Klik pada code cell dan ketik: `print("Bismillah, saya siap belajar Python!")`
   - Tekan `Shift+Enter` untuk menjalankan.
   - Output akan muncul di bawah cell.
6. **Menyimpan notebook:**
   - Notebook otomatis tersimpan di Google Drive Anda.
   - Untuk mengunduh: **File > Download > Download .ipynb**
7. **Mengunggah file:**
   - Klik ikon folder di panel kiri untuk membuka file browser.
   - Klik ikon upload untuk mengunggah file dari komputer.
   - Atau gunakan kode: `from google.colab import files; files.upload()`
8. **Mengunduh file hasil:**
   - Gunakan kode: `from google.colab import files; files.download('nama_file.txt')`

**Tips penggunaan Colab:**

- Runtime Colab akan terputus setelah tidak aktif selama 90 menit. Simpan pekerjaan Anda secara berkala.
- Colab menyediakan GPU/TPU gratis untuk komputasi berat (tidak diperlukan untuk kuliah ini).
- Shortcut penting: `Ctrl+Enter` (jalankan cell), `Shift+Enter` (jalankan cell dan pindah ke cell berikutnya), `Ctrl+M B` (tambah cell baru di bawah).

### D.2 Instalasi Python (Lokal)

Jika Anda ingin menjalankan Python di komputer lokal, ikuti langkah berikut.

**Unduh Python:**

1. Kunjungi https://www.python.org/downloads/
2. Klik tombol **"Download Python 3.x.x"** (versi terbaru).

**Instalasi di Windows:**

1. Jalankan file installer `.exe` yang sudah diunduh.
2. **PENTING:** Centang kotak **"Add Python to PATH"** di bagian bawah jendela installer.
3. Klik **"Install Now"** untuk instalasi standar.
4. Tunggu proses instalasi selesai, lalu klik **"Close"**.
5. Verifikasi instalasi — buka **Command Prompt** (ketik `cmd` di Start Menu) dan jalankan:
   ```
   python --version
   ```
   Output yang diharapkan: `Python 3.x.x`

**Instalasi di macOS:**

1. Jalankan file installer `.pkg` yang sudah diunduh.
2. Ikuti petunjuk pada wizard instalasi.
3. Verifikasi instalasi — buka **Terminal** dan jalankan:
   ```
   python3 --version
   ```
4. Catatan: macOS mungkin memiliki Python 2 bawaan. Selalu gunakan perintah `python3` untuk memastikan versi yang benar.

**Instalasi di Linux (Ubuntu/Debian):**

1. Python biasanya sudah terinstal. Verifikasi dengan:
   ```
   python3 --version
   ```
2. Jika belum terinstal:
   ```
   sudo apt update
   sudo apt install python3 python3-pip
   ```

### D.3 Visual Studio Code (VS Code)

VS Code adalah editor kode gratis dari Microsoft yang sangat populer untuk pengembangan Python.

**Langkah instalasi:**

1. Unduh VS Code dari https://code.visualstudio.com/
2. Instal sesuai sistem operasi Anda.
3. Buka VS Code, lalu instal ekstensi **Python**:
   - Klik ikon Extensions di sidebar kiri (atau tekan `Ctrl+Shift+X`).
   - Cari **"Python"** (oleh Microsoft).
   - Klik **"Install"**.
4. **Membuat dan menjalankan file Python:**
   - Buat file baru dengan ekstensi `.py`, misalnya `hello.py`.
   - Tulis kode: `print("Hello, World!")`
   - Klik tombol play (Run) di pojok kanan atas, atau tekan `Ctrl+F5`.
5. **Fitur berguna di VS Code:**
   - **IntelliSense**: Auto-complete dan saran kode.
   - **Linting**: Pengecekan kesalahan secara real-time.
   - **Integrated Terminal**: Terminal bawaan (`Ctrl+`` `).
   - **Debugging**: Pasang breakpoint dan jalankan debugger.

### D.4 PyCharm (Opsional)

PyCharm adalah IDE khusus Python yang dikembangkan oleh JetBrains. Edisi Community tersedia secara gratis.

**Langkah instalasi:**

1. Unduh **PyCharm Community Edition** dari https://www.jetbrains.com/pycharm/download/
2. Instal sesuai sistem operasi Anda.
3. Buka PyCharm dan buat proyek baru (**New Project**).
4. Pastikan interpreter Python sudah terdeteksi.
5. Buat file Python baru: klik kanan pada proyek > **New > Python File**.
6. Tulis kode dan jalankan dengan klik kanan > **Run** atau tekan `Shift+F10`.

> **Rekomendasi untuk mahasiswa baru:** Gunakan **Google Colab** sebagai lingkungan utama. Colab tidak memerlukan instalasi, dapat diakses dari perangkat apapun yang memiliki browser, dan menyimpan pekerjaan Anda secara otomatis di Google Drive. Setelah terbiasa, Anda dapat beralih ke VS Code atau PyCharm untuk pengembangan yang lebih lanjut.
