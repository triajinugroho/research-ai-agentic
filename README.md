## Tentang Repository Ini

Repository ini berisi **materi kuliah lengkap** untuk mata kuliah Analisis Data Statistik yang dirancang khusus untuk mahasiswa Prodi Informatika UAI tingkat 1. Kurikulum ini mengintegrasikan pendekatan **OBE (Outcome-Based Education)** sesuai standar SN-Dikti/KKNI, dengan sentuhan modern: **AI-augmented data analysis** yang mempersiapkan mahasiswa untuk era data science dan artificial intelligence.

### Filosofi Desain

```
Fondasi Kuat + Tools Modern + AI Literacy = Future-Proof Data Analyst
```

- **Step-by-step**: Dari konsep dasar → inferensi → pemodelan → AI-augmented analysis
- **Practice-first**: Setiap teori langsung dipraktikkan dengan Python di Google Colab
- **Real-world**: Menggunakan dataset Indonesia (BPS, ekonomi digital, kesehatan)
- **AI-native**: AI bukan tambahan — tapi terintegrasi sebagai co-analyst sepanjang semester
- **Ethical**: Nilai-nilai Islami UAI tertanam dalam etika data dan responsible AI
