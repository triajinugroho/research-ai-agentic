### 5.2 Contoh Soal Code Tracing

**Soal CT-1:**
Tuliskan output dari kode berikut:
```python
def hitung_diskon(harga, persen):
    diskon = harga * persen / 100
    total = harga - diskon
    return total

barang = ["Buku", "Pensil", "Tas"]
harga_list = [50000, 5000, 150000]

for i in range(len(barang)):
    if harga_list[i] > 10000:
        bayar = hitung_diskon(harga_list[i], 10)
        print(f"{barang[i]}: Rp{int(bayar)}")
    else:
        print(f"{barang[i]}: Rp{harga_list[i]}")
```

**Jawaban:**
```
Buku: Rp45000
Pensil: Rp5000
Tas: Rp135000
```

**Soal CT-2:**
Tuliskan output dari kode berikut:
```python
kata = "UNIVERSITAS"
hasil = ""
for i in range(len(kata)):
    if i % 2 == 0:
        hasil += kata[i].lower()
    else:
        hasil += kata[i]
print(hasil)
print(len(hasil))
```

**Jawaban:**
```
uNiVeRsItAs
11
```

**Soal CT-3:**

### 5.3 Contoh Soal Tulis Kode

**Soal TK-1:**
Buatlah fungsi `hitung_bmi(berat, tinggi_cm)` yang menerima berat badan (kg) dan tinggi badan (cm), kemudian mengembalikan kategori BMI sebagai string:
- BMI < 18.5: "Kurus"
- 18.5 <= BMI < 25: "Normal"
- 25 <= BMI < 30: "Gemuk"
- BMI >= 30: "Obesitas"

Formula BMI = berat / (tinggi_m)^2, di mana tinggi_m = tinggi_cm / 100.

**Contoh Jawaban:**
```python
def hitung_bmi(berat, tinggi_cm):
    tinggi_m = tinggi_cm / 100
    bmi = berat / (tinggi_m ** 2)
    if bmi < 18.5:
        return "Kurus"
    elif bmi < 25:
        return "Normal"
    elif bmi < 30:
        return "Gemuk"
    else:
        return "Obesitas"
```

**Soal TK-2:**
Buatlah program yang meminta pengguna memasukkan sejumlah nilai mahasiswa (diakhiri dengan input -1), kemudian menampilkan rata-rata, nilai tertinggi, dan nilai terendah.

**Soal TK-3:**
Buatlah fungsi `cek_palindrom(kata)` yang menerima sebuah string dan mengembalikan `True` jika string tersebut adalah palindrom (dibaca sama dari depan dan belakang, abaikan huruf besar/kecil), dan `False` jika bukan.

## 7. Tips untuk Mahasiswa

1. **Latihan code tracing**: Biasakan membaca kode dan menuliskan output secara manual tanpa komputer
2. **Hafal method string dan list**: `.upper()`, `.lower()`, `.split()`, `.join()`, `.append()`, `.sort()`, `.index()`, dsb.
3. **Pahami scope variabel**: Bedakan variabel lokal dalam fungsi dan variabel global
4. **Perhatikan indentasi**: Dalam ujian tertulis, pastikan indentasi jelas dengan spasi yang konsisten
5. **Kelola waktu**: PG (15 menit) -> Code Tracing (25 menit) -> Tulis Kode (40 menit) -> Essay (20 menit)
6. **Periksa kembali**: Sisakan 5-10 menit terakhir untuk memeriksa jawaban
