### 3. Tabel Ringkasan Algoritma

| Algoritma | Kategori | Kompleksitas Waktu (Worst) | Kompleksitas Ruang | Kapan Digunakan |
|-----------|----------|---------------------------|-------------------|-----------------|
| Linear Search | Searching | O(n) | O(1) | Data belum terurut, ukuran data kecil |
| Binary Search | Searching | O(log n) | O(1) | Data sudah terurut, ukuran data besar |
| Bubble Sort | Sorting | O(n^2) | O(1) | Pembelajaran, data sangat kecil |
| Selection Sort | Sorting | O(n^2) | O(1) | Jumlah penukaran minimal penting |
| Insertion Sort | Sorting | O(n^2) | O(1) | Data hampir terurut, data kecil |
| Merge Sort | Sorting | O(n log n) | O(n) | Data besar, stabilitas diperlukan |
| Stack (push/pop) | Struktur Data | O(1) | O(n) | Undo/redo, validasi kurung, DFS |
| Queue (enq/deq) | Struktur Data | O(1) | O(n) | Antrian proses, BFS, penjadwalan |

### 4. Tips Persiapan UAS

**Strategi belajar:**

1. **Review catatan dan modul** -- Baca ulang ringkasan tiap minggu, fokus pada konsep yang belum dikuasai
2. **Latihan code tracing** -- Ambil kode dari modul, trace eksekusinya secara manual tanpa menjalankan komputer
3. **Tulis kode di kertas** -- Biasakan menulis kode Python tanpa bantuan IDE atau autocomplete
4. **Latihan soal** -- Kerjakan contoh soal di bawah dan soal-soal dari tugas-tugas sebelumnya
5. **Buat lembar catatan** -- Rangkum rumus, sintaks penting, dan tabel kompleksitas dalam 1 halaman A4

**Kesalahan umum yang harus dihindari:**

- Lupa `return` dalam fungsi (fungsi mengembalikan `None`)
- Off-by-one error pada indexing dan loop
- Mengubah list saat sedang melakukan iterasi
- Tidak menangani kasus kosong (empty list, empty string)
- Salah menentukan kompleksitas karena mengabaikan loop tersembunyi

### 5. Contoh Soal Latihan UAS

**Soal 1 (Pilihan Ganda -- Konsep):**
Manakah yang BUKAN pilar Computational Thinking?
a) Decomposition  b) Pattern Recognition  c) Compilation  d) Abstraction

**Soal 2 (Pilihan Ganda -- Kompleksitas):**
Berapa kompleksitas waktu worst-case dari Binary Search?
a) O(1)  b) O(n)  c) O(log n)  d) O(n^2)

**Soal 3 (Code Tracing):**
Tentukan output dari kode berikut:

```python
def misteri(n):
    if n <= 1:
        return n
    return misteri(n - 1) + misteri(n - 2)

print(misteri(6))
```

**Soal 4 (Code Tracing):**
Tentukan output dari kode berikut:

```python
data = [5, 3, 8, 1, 9, 2]
stack = []
for item in data:
    while stack and stack[-1] < item:
        stack.pop()
    stack.append(item)
print(stack)
```

**Soal 5 (Code Tracing):**
Tentukan output dari kode berikut:

```python
teks = "algoritma"
hasil = {}
for huruf in teks:
    hasil[huruf] = hasil.get(huruf, 0) + 1
print(sorted(hasil.items(), key=lambda x: x[1], reverse=True)[:3])
```

**Soal 6 (Tulis Kode):**
Tulis fungsi `cari_dua_terbesar(data)` yang menerima list bilangan bulat dan mengembalikan tuple berisi dua elemen terbesar. Tidak boleh menggunakan `sort()` atau `sorted()`. Sertakan validasi input dan docstring.

**Soal 7 (Tulis Kode):**
Tulis fungsi `is_palindrome_stack(teks)` yang menggunakan stack (implementasi list) untuk memeriksa apakah sebuah string adalah palindrome. Abaikan spasi dan huruf besar/kecil.

**Soal 8 (Tulis Kode):**
Tulis fungsi `merge_sorted_lists(list1, list2)` yang menggabungkan dua list yang sudah terurut menjadi satu list terurut tanpa menggunakan `sort()` atau `sorted()`. Analisis kompleksitas waktu dan ruangnya.

**Soal 9 (Essay/Analisis):**
Diberikan sebuah masalah: mencari apakah terdapat dua elemen dalam sebuah list yang jumlahnya sama dengan target tertentu.
a) Jelaskan pendekatan brute-force dan tentukan kompleksitasnya
b) Jelaskan pendekatan yang lebih efisien menggunakan dictionary/set
c) Bandingkan trade-off waktu dan ruang dari kedua pendekatan

**Soal 10 (Essay/Analisis):**
Jelaskan perbedaan antara stack dan queue. Untuk masing-masing, berikan satu contoh penerapan di dunia nyata dan satu contoh penerapan dalam pemrograman. Gambarkan ilustrasi operasi push/pop untuk stack dan enqueue/dequeue untuk queue.

### 6. Kunci Jawaban Ringkas Soal Pilihan Ganda dan Code Tracing
