## Tentang Repository Ini

Repository ini berisi **materi kuliah lengkap** untuk mata kuliah Kecerdasan Buatan dan Machine Learning yang dirancang khusus untuk mahasiswa Prodi Informatika UAI tingkat 3 (Semester 5). Kurikulum ini mengintegrasikan pendekatan **OBE (Outcome-Based Education)** sesuai standar SN-Dikti/KKNI, dengan fokus ganda: **fondasi AI/ML yang kuat** dan **applied machine learning** yang mempersiapkan mahasiswa untuk membangun sistem cerdas secara bertanggung jawab.

### Filosofi Desain

```text
Fondasi Kuat + Algoritma ML + Applied AI + AI Literacy = Future-Proof AI Engineer
```

- **Theory-to-practice**: Dari konsep dasar AI → supervised learning → deep learning → applied AI
- **Hands-on**: Setiap teori langsung dipraktikkan dengan Python (scikit-learn, TensorFlow/Keras) di Google Colab
- **Real-world**: Menggunakan dataset Indonesia (BPS, e-commerce, Open Data Jakarta)
- **AI-native**: AI bukan hanya materi — tapi terintegrasi sebagai coding partner sepanjang semester
- **Ethical**: Nilai-nilai Islami UAI tertanam dalam etika AI dan responsible ML
