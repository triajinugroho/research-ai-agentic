## Contoh Soal per Tipe

### A. Pilihan Ganda (PG) — 10 soal @ 1.5 poin

**Contoh Soal 1:**
> Dalam algoritma K-Means clustering, metode elbow digunakan untuk:
>
> A. Menentukan fitur mana yang paling penting
> B. Menentukan jumlah cluster optimal berdasarkan inertia
> C. Menghitung jarak antar data point
> D. Menentukan initial centroid
>
> **Jawaban: B** — Metode elbow memplot jumlah cluster (k) vs inertia (within-cluster sum of squares). Titik "siku" menunjukkan k optimal di mana penambahan cluster tidak lagi memberikan penurunan inertia yang signifikan.

**Contoh Soal 2:**
> Pada Convolutional Neural Network (CNN), operasi pooling (max pooling) berfungsi untuk:
>
> A. Menambah jumlah parameter model
> B. Melakukan augmentasi data
> C. Mengurangi dimensi spasial feature map dan menambah robustness terhadap translasi kecil
> D. Menghitung gradien untuk backpropagation
>
> **Jawaban: C** — Max pooling mengurangi dimensi spasial (width x height) dari feature map, mengurangi jumlah parameter, dan membuat model lebih robust terhadap pergeseran kecil pada posisi fitur.

**Contoh Soal 3:**
> Seorang data scientist membangun model prediksi kredit untuk bank di Indonesia. Model menunjukkan accuracy 95% tetapi hanya memprediksi 20% nasabah gagal bayar dengan benar. Masalah utama pada model ini adalah:
>
> A. Model terlalu kompleks (overfitting)
> B. Dataset imbalanced sehingga model bias terhadap kelas mayoritas
> C. Fitur yang digunakan tidak relevan
> D. Learning rate terlalu tinggi
>
> **Jawaban: B** — Accuracy tinggi dengan recall rendah pada kelas minoritas (gagal bayar) menunjukkan dataset imbalanced. Model cenderung memprediksi kelas mayoritas (tidak gagal bayar) untuk mendapatkan accuracy tinggi.

### B. Short Answer — 4 soal @ 5 poin

**Contoh Soal 1:**
> Jelaskan apa itu PCA (Principal Component Analysis). Bagaimana menentukan jumlah komponen utama yang optimal? Apa keuntungan dan keterbatasan PCA? (5 poin)
>
> **Jawaban:**
> - **PCA** adalah teknik reduksi dimensi yang mentransformasi data ke sistem koordinat baru berdasarkan arah variansi terbesar (principal components). PC1 menjelaskan variansi terbesar, PC2 menjelaskan variansi terbesar berikutnya (ortogonal terhadap PC1), dst.
> - **Jumlah komponen optimal:** Gunakan explained variance ratio. Pilih jumlah komponen yang menjelaskan cukup variansi (misal >= 95% cumulative explained variance).
> - **Keuntungan:** Mengurangi dimensi, menghilangkan multikolinearitas, mempercepat training.
> - **Keterbatasan:** Komponen sulit diinterpretasi, mengasumsikan linearitas, sensitif terhadap skala (perlu scaling dulu).

**Contoh Soal 2:**
> Jelaskan proses backpropagation pada neural network. Mengapa dibutuhkan fungsi aktivasi non-linear? (5 poin)
>
> **Jawaban:**
> - **Backpropagation:** Setelah forward pass menghitung prediksi dan loss, backpropagation menghitung gradien loss terhadap setiap weight menggunakan chain rule. Gradien ini digunakan untuk memperbarui weight (gradient descent) agar loss berkurang.
> - **Fungsi aktivasi non-linear:** Tanpa non-linearitas, jaringan multi-layer hanya ekuivalen dengan satu transformasi linear (komposisi fungsi linear = fungsi linear). Fungsi non-linear (ReLU, sigmoid, tanh) memungkinkan network mempelajari hubungan non-linear dalam data.

### D. Tulis Kode — 2-3 soal @ ~10 poin

**Contoh Soal 1:**
> Tuliskan kode Python lengkap untuk melakukan K-Means clustering pada dataset, menentukan jumlah cluster optimal menggunakan elbow method, dan menghitung silhouette score. Asumsikan data `X` sudah tersedia dan sudah di-scaling. (10 poin)
>
> **Jawaban:**
> ```python
> from sklearn.cluster import KMeans
> from sklearn.metrics import silhouette_score
> import matplotlib.pyplot as plt
>
> # Elbow method - mencari k optimal
> inertias = []
> sil_scores = []
> K_range = range(2, 11)
>
> for k in K_range:
>     kmeans = KMeans(n_clusters=k, random_state=42, n_init=10)
>     kmeans.fit(X)
>     inertias.append(kmeans.inertia_)
>     sil_scores.append(silhouette_score(X, kmeans.labels_))
>
> # Plot Elbow
> plt.figure(figsize=(12, 4))
> plt.subplot(1, 2, 1)
> plt.plot(K_range, inertias, 'bo-')
> plt.xlabel('Jumlah Cluster (k)')
> plt.ylabel('Inertia')
> plt.title('Elbow Method')
>
> # Plot Silhouette Score
> plt.subplot(1, 2, 2)
> plt.plot(K_range, sil_scores, 'ro-')
> plt.xlabel('Jumlah Cluster (k)')
> plt.ylabel('Silhouette Score')
> plt.title('Silhouette Score per k')
> plt.tight_layout()
> plt.show()
>
> # Pilih k optimal (misal k=3) dan tampilkan hasil
> k_optimal = K_range[sil_scores.index(max(sil_scores))]
> print(f"K optimal berdasarkan silhouette: {k_optimal}")
> print(f"Silhouette score: {max(sil_scores):.4f}")
> ```

**Contoh Soal 2:**
> Tuliskan kode Python untuk membangun pipeline sentiment analysis sederhana pada teks Bahasa Indonesia. Pipeline harus mencakup: TF-IDF vectorization, training model Naive Bayes, dan evaluasi. Asumsikan `texts_train`, `y_train`, `texts_test`, `y_test` sudah tersedia. (10 poin)
>
> **Jawaban:**
> ```python
> from sklearn.feature_extraction.text import TfidfVectorizer
> from sklearn.naive_bayes import MultinomialNB
> from sklearn.pipeline import Pipeline
> from sklearn.metrics import classification_report
>
> # Buat pipeline: TF-IDF + Naive Bayes
> pipeline = Pipeline([
>     ('tfidf', TfidfVectorizer(max_features=5000)),
>     ('clf', MultinomialNB())
> ])
>
> # Training
> pipeline.fit(texts_train, y_train)
>
> # Prediksi
> y_pred = pipeline.predict(texts_test)
>
> # Evaluasi
> print("=== Hasil Evaluasi Sentiment Analysis ===")
> print(classification_report(y_test, y_pred,
>       target_names=['Negatif', 'Positif']))
> ```

### E. Essay/Desain — 2 soal @ 7.5 poin

**Contoh Soal 1:**
> **Skenario:** Sebuah rumah sakit di Jakarta ingin membangun sistem AI untuk membantu diagnosis awal penyakit berdasarkan gejala pasien. Data yang tersedia meliputi: usia, jenis kelamin, tekanan darah, kadar gula, riwayat penyakit keluarga, gejala (teks bebas), dan diagnosis dokter (5 kategori penyakit).
>
> a. Rancang pipeline ML end-to-end untuk sistem ini, mulai dari data collection hingga deployment. Jelaskan setiap tahap. (3 poin)
> b. Identifikasi minimal 3 risiko etika dari sistem ini dan bagaimana mengatasinya. (2.5 poin)
> c. Bagaimana memastikan model ini adil (fair) untuk semua kelompok pasien? (2 poin)
>
> **Jawaban:**
> a. **Pipeline ML:**
>    1. **Data Collection & Cleaning:** Kumpulkan data pasien (anonim), bersihkan missing values, standarisasi format
>    2. **Feature Engineering:** Encode kategorikal (gender, riwayat), scale numerik (usia, tekanan darah), TF-IDF untuk gejala teks
>    3. **Model Selection:** Coba multiple models (Random Forest, SVM, Neural Network) karena multi-class classification
>    4. **Evaluation:** Gunakan stratified k-fold CV, fokus pada recall per kelas (jangan sampai miss diagnosis)
>    5. **Deployment:** API endpoint dengan monitoring prediksi, feedback loop dari dokter
>
> b. **Risiko etika:**
>    1. **Bias data:** Dataset mungkin over-represent pasien tertentu (usia, gender). Solusi: audit distribusi data, stratified sampling
>    2. **False negative berbahaya:** Model salah memprediksi "sehat" padahal sakit. Solusi: optimasi recall, threshold yang konservatif
>    3. **Privasi data pasien:** Data medis sangat sensitif. Solusi: enkripsi, akses terbatas, compliance UU PDP
>
> c. **Fairness:** Evaluasi performa model per subgrup (usia, gender, etnis). Gunakan metrik fairness (demographic parity, equal opportunity). Jika ada disparitas, re-balance training data atau gunakan fairness-aware algorithms.

**Contoh Soal 2:**
> **Skenario:** Kementerian Pendidikan ingin menggunakan AI untuk memprediksi siswa yang berisiko putus sekolah di seluruh Indonesia. Anda diminta merancang solusi ML yang bertanggung jawab.
>
> a. Jelaskan 3 jenis bias yang mungkin muncul dalam dataset dan model ini. (3 poin)
> b. Bagaimana Anda akan memvalidasi bahwa model tidak diskriminatif terhadap kelompok tertentu? (2.5 poin)
> c. Sebagai seorang Muslim, nilai-nilai apa yang harus dipegang dalam mengembangkan sistem AI ini? (2 poin)
>
> **Jawaban:**
> a. **Jenis bias:**
>    1. **Historical bias:** Data historis mungkin mencerminkan diskriminasi yang sudah ada (misal: daerah tertentu under-reported)
>    2. **Representation bias:** Siswa dari daerah terpencil mungkin under-represented dalam dataset
>    3. **Measurement bias:** Indikator "risiko putus sekolah" mungkin berbeda konteks di perkotaan vs pedesaan
>
> b. **Validasi fairness:** Hitung metrik performa (accuracy, recall) per subgrup (wilayah, gender, status ekonomi). Pastikan equal opportunity: recall harus setara antar kelompok. Gunakan fairness dashboard untuk monitoring.
>
> c. **Nilai Islam:** Amanah (menjaga data siswa dengan penuh tanggung jawab), Adalah (keadilan — model harus adil untuk semua siswa), Ihsan (berbuat baik — tujuan model adalah membantu, bukan menghakimi).

    |-- Data besar + kompleks --> Ya (Neural Networks)
        |-- Teks --> RNN, LSTM, Transformer
        |-- Gambar --> CNN
        |-- Tabular --> Masih bisa ML tradisional
```
