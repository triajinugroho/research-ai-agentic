## Filosofi

Praktikum ini merupakan komponen hands-on dari MK Rekayasa Perangkat Lunak (IF2205). Mahasiswa menerapkan konsep-konsep teori rekayasa perangkat lunak ke dalam praktik pengembangan aplikasi web (*web application development*) menggunakan Python (Flask), JavaScript (HTML/CSS/JS), dan tools industri seperti Git, Docker, dan GitHub Actions di lingkungan GitHub Codespaces.
