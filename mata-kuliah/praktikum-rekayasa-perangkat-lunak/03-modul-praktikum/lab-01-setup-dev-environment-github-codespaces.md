## Tujuan Praktikum

Setup GitHub account dan Codespace, install Python/Flask, buat Hello World

## Persiapan

- GitHub Codespaces terkonfigurasi
- Repository proyek tim
- Materi minggu ini sudah dibaca

## Langkah-langkah

1. Buat akun GitHub dan repository baru
2. Buka Codespace dari repository
3. Install Flask: pip install flask
4. Buat app.py dengan route Hello World
5. Jalankan: flask run

## Tantangan Tambahan

1. Tambahkan route /about
2. Dokumentasikan proses dan hasil di laporan praktikum
