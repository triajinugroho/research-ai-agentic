## Tentang Repository Ini

Repository ini berisi **materi kuliah lengkap** untuk mata kuliah Rekayasa Perangkat Lunak yang dirancang khusus untuk mahasiswa Prodi Informatika UAI tingkat 2 (Semester 4). Kurikulum ini mengintegrasikan pendekatan **OBE (Outcome-Based Education)** sesuai standar SN-Dikti/KKNI, dengan fokus ganda: **fondasi rekayasa perangkat lunak yang kuat** dan **praktik modern SE** yang mempersiapkan mahasiswa untuk membangun perangkat lunak berkualitas secara profesional dan bertanggung jawab.

### Filosofi Desain

```text
Fondasi SE + SDLC Modern + Agile/DevOps + AI-Augmented Development = Future-Proof Software Engineer
```

- **Full-stack SE Process**: Dari requirements → design → construction → testing → deployment → maintenance
- **Hands-on**: Setiap teori langsung dipraktikkan dengan Python + JavaScript di GitHub Codespaces
- **Real-world**: Menggunakan proyek konteks Indonesia (UMKM, layanan publik, kampus)
- **AI-native**: AI tools (Copilot, Claude Code, Cursor) terintegrasi sebagai coding partner sepanjang semester
- **Ethical**: Nilai-nilai Islami UAI tertanam dalam etika profesi dan responsible software development
