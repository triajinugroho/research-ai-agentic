## Prinsip Pemilihan Resources

Resources untuk mata kuliah ini dipilih berdasarkan kriteria:
1. **Relevan dengan industri** — Tools dan platform yang digunakan di industri software Indonesia
2. **Gratis atau freemium** — Mahasiswa bisa mengakses tanpa biaya
3. **Cloud-based** — Dapat digunakan tanpa instalasi berat di lokal
4. **Konteks Indonesia** — Contoh proyek dan studi kasus lokal
5. **Modern & terkini** — Sesuai praktik SE 2025-2026
