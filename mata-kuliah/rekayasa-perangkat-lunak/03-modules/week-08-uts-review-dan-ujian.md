### Contoh Soal Review

**Pilihan Ganda (C2):**
Manakah yang BUKAN merupakan Scrum Event?
- a) Sprint Planning
- b) Daily Standup
- c) Code Review ✓
- d) Sprint Retrospective

**Uraian Singkat (C2-C3):**
Jelaskan perbedaan antara Functional Requirements dan Non-Functional Requirements. Berikan masing-masing 2 contoh untuk Sistem Informasi Perpustakaan.

**Studi Kasus (C3-C4):**
Sebuah startup e-commerce di Jakarta ingin membangun platform marketplace. Tim terdiri dari 5 developer, deadline 6 bulan, requirements bisa berubah.
1. Model proses apa yang paling cocok? Jelaskan alasannya.
2. Buatlah 3 user stories untuk fitur utama.
3. Gambarkan arsitektur high-level yang sesuai.

**Analisis Kode (C4):**
Identifikasi 3 code smells dalam kode berikut dan jelaskan bagaimana cara refactoring-nya:
```python
def process_order(order_data, user_data, product_data, payment_data):
    # ... (200 baris kode dengan multiple responsibilities)
```
