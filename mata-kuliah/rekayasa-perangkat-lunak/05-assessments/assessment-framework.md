## Kebijakan Integritas Akademik

1. **Plagiarisme**: Copy-paste tanpa atribusi → nilai 0 untuk komponen tersebut
2. **Penggunaan AI saat ujian**: Menggunakan AI saat kuis/UTS/UAS → nilai 0 + laporan ke prodi
3. **Kolaborasi yang tidak diizinkan**: Tugas individu harus dikerjakan sendiri
4. **Self-plagiarism**: Menggunakan tugas dari mata kuliah lain tanpa izin
